## Snapshot

Limits at infinity describe **end behavior**. If $f(x)$ approaches a finite $L$
as $x\to\pm\infty$, then $y=L$ is a horizontal asymptote. Dominant terms govern
polynomials/rationals as $|x|\to\infty$.

## Formal Statement

We write
```tex
\lim_{x\to +\infty} f(x) = L \quad\text{or}\quad \lim_{x\to -\infty} f(x) = L
```
to capture stabilization of $f(x)$ for large $|x|$.

A **horizontal asymptote** is
```tex
y=L \quad \text{if} \quad \lim_{x\to\pm\infty} f(x)=L.
```

Dominant-term rules (rationals):

- Degrees equal → limit = ratio of leading coefficients.  
- Numerator degree < denominator → limit $=0$.  
- Numerator degree > denominator → $|f(x)|\to\infty$ (no horizontal asymptote).

## Intuition & Interpretation

Local irregularities vanish at large $|x|$; only leading structure matters.

## Examples & Non-Examples

### Examples

1.
```tex
\lim_{x\to\infty} \frac{3x^2 - x}{6x^2 + 1} = \frac{3}{6} = \frac{1}{2}.
```

2.
```tex
\lim_{x\to\infty} \frac{x}{x^2 + 1} = 0.
```

3.
```tex
\lim_{x\to -\infty} \frac{x^3 + 1}{2x^2 - 5} = -\infty.
```

4.
```tex
\lim_{x\to\infty} \frac{x^2}{e^x} = 0.
```

### Non-Examples

1. **Oscillation**
```tex
\lim_{x\to\infty} \sin x \ \text{does not exist.}
```
2. **Finite‑point infinite limit** — That’s not a limit at infinity.  
3. **Piecewise with non‑stabilizing behavior** — No horizontal asymptote.

## Properties & Relationships

- Dominant terms drive polynomials/rationals.  
- Exponential/logarithmic/polynomial families have predictable dominance.  
- Closely related to growth rates.

## Common Pitfalls

1. Assuming a function cannot cross its horizontal asymptote.  
2. Applying finite‑point rules without dominance checks.  
3. Forgetting sign when $x\to -\infty$.  
4. Misidentifying degrees.

## Exercises

1. Compute
```tex
\lim_{x\to\infty} \frac{5x^3 - x}{10x^3 + 2}.
```

2. Dominant‑term check
```tex
\lim_{x\to-\infty} \frac{7x^2}{3x^3 -1}.
```

3. Conceptual — Why does denominator degree one higher imply asymptote $y=0$?

4. Mini‑proof — If $\lim_{x\to\infty} f(x)=L$, explain why $y=L$ is a horizontal asymptote.
